### 下载

从 xmindchina 官方网站下载：

https://www.xmind.cn/download/

Xmind 支持 windows，mac 和 linux，下载 linux 64位版本即可，如 ` XMind-for-Linux-amd-64bit-11.0.1-202106220606.deb`  即可。


### 安装

直接安装下载好的` XMind-for-Linux-amd-64bit-11.0.1-202106220606.deb` 就可以了.
