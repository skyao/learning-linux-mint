## 介绍

我用来进行文件同步的工具，支持 windows / linux / macos。

## 安装

下载地址： 

https://freefilesync.org/download.php

解压缩下载下来的 FreeFileSync_13.8_Linux.tar.gz 文件，得到 FreeFileSync_13.8_Install.run 文件，执行 

```bash
./FreeFileSync_13.8_Install.run
```

安装即可。

用法和windows一致。
