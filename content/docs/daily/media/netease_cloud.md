最喜欢的在线音乐播放器了, 之前没有linux版本，只能通过 wine 来安装，后来网易终于出了 linux 版本，开心 :)

## 下载

网易云音乐的下载地址：

http://music.163.com/download

选择 "Linux版"，在弹出窗口列出的多个版本中，选择"ubuntu 16.04 64位" 版本下载。

## 安装

直接用 GDebi package installer 安装即可，感谢网易和deepin，现在在linux上用网易云音乐轻松多了。
