System Monitor 是 Linux Mint 自带的系统监控工具，在 "开始菜单" -> "系统管理" -> "System Monitor" 打开。

1. Processes 看进程信息，相当于图形版本的 top
2. Resources 看资源信息
3. File Systems 看硬盘分区和分区使用情况
